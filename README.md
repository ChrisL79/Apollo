# Apollo
Apollo is a network PTT program for SunSDR radios over TCI with user selectablepre and/or post Quindar tones from the Gemini/Apollo era. Apollo will inject the selected mic directly into the TX audio chain. I use it for transmitting on QO-100 satellite.
